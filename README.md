# 校边 AI 小店参谋

这是一个用于笔试提交的轻量机会验证项目，方向选择为“小微 AI”。

## 文件说明

- `submission-a4.html`：一页 A4 提交稿，可直接打印或导出 PDF。
- `submission-a4.pdf`：已导出的 A4 PDF，用于上传飞书表单。
- `prototype/index.html`：可交互 MVP 原型，双击即可在浏览器打开。
- `prototype/styles.css`：原型页面样式。
- `prototype/app.js`：原型交互逻辑，输入店铺信息后生成内容包。
- `prompts.md`：真实使用 AI 生成内容包时可复制的 Prompt。

## 如何提交

1. 优先上传 `submission-a4.pdf`。
2. 如表单允许附链接或压缩包，可以同时上传本文件夹或附上 `prototype/index.html` 截图。
3. 如果需要演示 MVP，双击打开 `prototype/index.html`，输入店铺信息后点击“生成 7 天内容包”。

## 验证假设

如果两周内能找到 3 家学校周边商家试用，并且至少 1 家愿意为 99/299 元套餐付费，则说明该机会值得继续推进。
